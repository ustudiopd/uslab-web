## ❌ 미구현 기능

### 🔴 높은 우선순위 (명세서 필수 항목)

#### 1. Dashboard 차트
- ❌ **트래픽 추이 차트** (최근 7/30일 일별 Pageviews)
  - 데이터는 `getDailyStats()` 함수로 준비됨
  - 차트 라이브러리 필요 (recharts 권장)
  - 7일/30일 토글 필터
- ❌ **SEO 상태 박스**
  - 기술적 SEO 체크 (sitemap/robots/canonical/JSON-LD 존재 여부)
  - 포스트 SEO 품질 (누락 개수 집계)

#### 2. SEO 기술적 구현
- ❌ **`app/sitemap.ts`**
  - 발행된 포스트 전체 + 언어별 URL
  - lastmod, priority 포함
- ❌ **`app/robots.ts`**
  - Sitemap 위치 명시
  - `/admin/*`, `/api/*` 크롤링 차단
- ❌ **Canonical URL 명시적 추가**
  - 현재는 hreflang만 있음
  - `<link rel="canonical">` 태그 추가 필요
- ❌ **JSON-LD 구조화된 데이터**
  - Organization 스키마 (site-wide)
  - Article 스키마 (블로그 포스트)
  - BreadcrumbList 스키마 (블로그 상세)

#### 3. Admin 페이지
- ❌ **`/admin/comments`** (댓글 관리)
  - 리스트: post, author_name, created_at, is_approved
  - 액션: 승인 토글, 삭제
  - 필터: post_id, 날짜
- ❌ **`/admin/inquiries`** (문의 관리)
  - 리스트: name, email, status, created_at
  - 디테일: message 전문
  - 액션: status 변경 (pending/contacted/completed)
- ❌ **`/admin/analytics`** (트래픽 상세)
  - 일별/주별/월별 상세 통계
  - 디바이스별 분석
  - Referrer 상세 분석
- ❌ **`/admin/seo`** (SEO 설정/감사)
  - SEO 설정 페이지
  - `/admin/seo/audit` (포스트 SEO 품질 검사)
    - seo_title 누락 개수
    - seo_description 누락 개수
    - thumbnail_url 누락 개수
    - title/description 길이 초과 개수

#### 4. Post 편집 화면 SEO 패널 강화
- ❌ **SEO 편집 패널**
  - seo_title/seo_description/seo_keywords 편집
  - 글자수 카운터 (60/160 권장)
  - "AI로 재생성" 버튼 (이미 `/api/ai/seo` 있음)
  - OG 미리보기 (썸네일 기반)
